HTML

During the first couple of days we've learned about HTML and practice by creating projects with exercises. HTML is hypertext markup language that structures a webpage with meaningful information for various manupilation for styling and content delivery via backend database. We are achieving this in HTML via tags. Each HTML page starts with Doctype, html tag and head where title, meta data, link to CSS files then body tag where main content of a page resides. example below. 

    
    
        <!Doctype html>
        <html>
            <head>
                <title>This is Title</title>
            </head>
            <body>
                <p>This is content</p>
            </body>
        </html>
CSS

CSS stands for Cascading Style Sheets and is a language to style an html page with various styles. The best practice for it is to use link tag to link external dot css file within page's file structure in html file ie index.html or any other html file that requires styling. then using dot css file using ids classes and tags with properties and values to style various components and its behavior in html page. example below: 

        
            .sidebar {
                width: 85px;
                position: fixed;
                z-index: 1; /*overlapping position on z axis on a page*/ 
                top: 30px; /*position from the top of the page */
                left: 10px; /*position from the left of the page */
                background: grey;
                overflow-x: hidden; /*hidden any overflow on x axis*/
                padding: 12px 2px;
                }
